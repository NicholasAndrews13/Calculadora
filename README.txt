1:Baixar o arquivo calculadora do github
2:Instalar o netbeans
3:No netbeans File=>Open Project ...
4:Selecionar a pasta calculadora
5:Expandir arquivo da calculadora, Source Packages e com.mycompany.calculadora
6:Clique duplo no arquivo Formulario.java
7:Realizar o comando shift + F6 para rodar o programa e é aberto uma calculadora que calcula 
operações basicas entre 2(dois) números.
